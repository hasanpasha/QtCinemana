## Usage
firstly download the source:
```
$ git clone https://github.com/hasanpasha/QtCinemana
$ cd cinemana
```
Now you need to get the reuqired packages:
```
$ pip install -r requirments.txt
```
To run the program:
```
$ python QtCinemana.py
```
